Statistical transformations

Lets consider the difference between geom_bar() and geom_point()

ggplot(mpg, aes(x = displ, y = hwy))+
  geom_point()

ggplot(mpg, aes(x = drv))+
  geom_bar()

geom_point(): data > plot

geom_bar(): data > counts > plot

Geom_bar()

geom_bar() has one more step between data and plot. It uses the data to calcuate the counts and then draw the plot based on the counts.

Many graphs, like scatterplots, plot the raw values of your dataset. Other graphs, like bar charts, calculate new values to plot.

Data transformation

To illustrate data transformation, we will use a new dataset diamonds

ggplot(data = diamonds) + 
  geom_bar(mapping = aes(x = cut))

  • On the x-axis, the chart displays cut, a variable from diamonds
  • On the y-axis, the chart displays count, calculated from data
  • The algorithm used to calculate new values for a graph is called a stat, short for statistical transformation

geoms and stats

  • You can generally use geoms and stats interchangably.
  • For example, you can recreate the previous plot using stat_count() instead of geom_bar
ggplot(data = diamonds) + 
  stat_count(mapping = aes(x = cut))

stats and geoms

  • The code from the previous slide works because every geom has a default stat; and every stat has a default geom
  • We can override the default mapping from transformed variables to aesthetics.
ggplot(data = diamonds) + 
  geom_bar(mapping = aes(x = cut, y = ..prop.., group = 1))

Summary statistics

  • In descriptive statistics, summary statistics are used to summarize a set of observations
    • Communicate the largest amount of information as simply as possible
  • There are two very important attributes that we try to describe
    • A measure of location or central tendency of the data
      • Mean, median
    • A measure of statistical dispersion or spread
      • Standard deviation, variance, range, interquartile range

Mean

The mean or average is the sum of a collection of numbers divided by the count of numbers in the collection.

a <- c(1,1,2,2,3,3,3,3,3,4,5,6)
b <- c(-3,2,4,2,5,-6,1,2,2,4,4,-2)
mean(a)
## [1] 3
mean(b)
## [1] 1.25

Medians

A median is a value separating the higher half from the lower half of a data sample

median(a)
## [1] 3
median(b)
## [1] 2

Quantiles

Quantiles are cut points dividing the sample into intervals with pre-specified proportions observations. Quartiles are the three cut points that will divide a dataset into four equal-sized groups.

quantile(a,probs=c(0.25,0.5,0.75))
##  25%  50%  75% 
## 2.00 3.00 3.25
quantile(b,probs=c(0.25,0.5,0.75))
##  25%  50%  75% 
## 0.25 2.00 4.00

Range

The range of a set of data is the difference between the largest and smallest values

range(a)
## [1] 1 6
range(b)
## [1] -6  5

Interquartile range

Interquartile range is the difference between the third and the first quartiles.

IQR(a)
## [1] 1.25
IQR(b)
## [1] 3.75

Variance and standard deviation

Variance is the expectation of the squared deviation of a random variable from its mean. It measures how far a set of numbers are spread out from their average value.

var(a)
## [1] 2.181818
var(b)
## [1] 10.93182
sd(a)
## [1] 1.477098
sd(b)
## [1] 3.30633

Summary

summary(a)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##    1.00    2.00    3.00    3.00    3.25    6.00
summary(b)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##   -6.00    0.25    2.00    1.25    4.00    5.00

Position adjustment

Position adjustments are used to adjust the position of each geom. It contols how overlapping objects are arranged. There are five different adjustments

  • “identity”: default of most geoms
  • “jitter”: default of geom_jitter()
  • “dodge” default of geom_boxplot()
  • “stack” default of geom_bar() and geom_histogram()
  • “fill”: useful for geom_bar() and geom_histogram()

Position adjustment

In a boxplot, if you map the fill aesthetic to another variable, like clarity: the bars are automatically stacked.

In the following plot, each colored rectangle represents a combination of cut and clarity

ggplot(data = diamonds) + 
  geom_bar(mapping = aes(x = cut, fill = clarity))

Position adjustment

The stacking is performed automatically by the position adjustment specified by the position argument.

ggplot(data = diamonds) + 
  geom_bar(mapping = aes(x = cut, fill = clarity), position = "fill")
  
ggplot(data = diamonds) + 
  geom_bar(mapping = aes(x = cut, fill = clarity), position = "dodge")
